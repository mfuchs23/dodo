    MessageAnil,

    > The utility that comes with dbdoclet uses Apache FOP which works just fine
    > but XEP dies on large input with an OutOfMemory error/exception.

    XEP's capability of handling large input files greatly depends on Java VM settings
    for heap size. If you don't modify the original scripts (xep.bat/xep.sh), the upper
    limit for memory available to Java is set to 64MB. Setting a larger value
    (via -Xmx/-Xms parameters in the call to Java machine) may improve the situation.
